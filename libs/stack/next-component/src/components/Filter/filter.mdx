# Filter

Filter component. Based on react-stately `useListState` wrapping `useQueryState`, it matches search params to the values selected in the `TagGroup`'s  `Popover`.

## Props

<Controls />

## Usage

### Default

```tsx
<Filter id="shape">
  <Item key="circle">Circle</Item>
  <Item key="square">Square</Item>
  <Item key="triangle">Triangle</Item>
</Filter>
```

### Controlled

```tsx
const [selectedKeys, setSelectedKeys] = useState<Key[]>([])

<Filter id="shape" selectedKeys={selectedKeys} onSelectionChange={setSelectedKeys}>
  <Item key="circle">Circle</Item>
  <Item key="square">Square</Item>
  <Item key="triangle">Triangle</Item>
</Filter>
```
