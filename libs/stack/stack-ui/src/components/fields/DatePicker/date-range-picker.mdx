# Date Range Picker

Date Range Picker component. As opposed to the `DatePicker` component, this component allows the user to select a range two dates.

### Modified behaviors

#### Unavailable dates

In this example, unavailable dates will be rendered as disabled, making the user unable to select them. However, they are still technically selectable via the date field.

If the user attempts to select an unavailable date via the date field, the `isInvalid` state will be applied to the whole component as a token.
