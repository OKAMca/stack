## Subcomponents

### ListBox

The component managing the list of items and the selection relative to that list. In the case of a ComboBox, the list is the list of items get filtered to match the input value.

The combo box's list box component only appears when the popover is open.

Documentation about the ListBox component can be found [here](?path=/docs/form-fields-listbox--docs).

### Popover

The component wrapping the list box and deciding when to open or close itself.

Documentation about the Popover component can be found [here](?path=/docs/base-components-popover--docs).
