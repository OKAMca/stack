# Side Panel

The side panel component is a modal component that opens from the side of the screen. It requires to be called within a `SidePanelContextProvider` to work. This allows for controlled behavior for components inside the side panel.

Use `useSidePanel` hook to get the `buttonProps`, `overlayProps` and `overlayState` props to control the side panel's open state.

### Controlled

You can pass the `isOpen` prop and state its state in the `onOpenChange` prop to control the side panel's open state.

```tsx
const [isOpen, setIsOpen] = useState(false)

<SidePanelContextProvider isOpen={isOpen} onOpenChange={setIsOpen}>
  <SidePanel>Hello world!</SidePanel>
</SidePanelContextProvider>
```
